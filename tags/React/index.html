React Bind

5月 14, 2016

React on ES6 不再自動綁定，這是在很多網路文章都會提到的事，但總是覺得矇矇懂，今天看到一個範例終於比較有感了。

就是在之前 handleOptionsButtonClick 內的 this 不需要綁定，就會預設為 this 是一個 component instance，但是在 ES6 的寫法，你需要加 this.handleOptionsButtonClick.bind(this)，它才會將 handleOptionsButtonClick 內的 this 視為是 component instance，否則使用 component instance 的 method。
