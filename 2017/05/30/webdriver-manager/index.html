推薦用 webdriver manager

Posted at 2017-05-30 webdriver-manager 

最近踩到一個 selenium-standalone 的雷，似乎 selenium-standalone 沒有及時 Sync 到最新版的 driver，Selenium 官網已經有新版的 Driver release，用 selenium-standalone 安裝取到的就是前一版，導致 Firefox 無法正常執行前端測試。

推薦 webdriver-manager 取代 selenium-standalone。
