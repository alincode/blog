PM2 管理工具

4月 03, 2016

它主要的功能是保證你的 process 永遠活著，如果有意外關閉，它會自動幫你重開，簡易管理多個程序。

安裝

npm install pm2@latest -g

常用指令

啟動程序

pm2 start app.js

啟動程序並指定名稱

pm2 start app.js --name my-api

啟動程序並帶變數

NODE_ENV=local pm2 start app.js --name my-api

查看所有註冊的程序

pm2 list

查看程序內容

pm2 show 0

監看程序

pm2 monit

查看log集合

pm2 logs

清空log

pm2 flush

Actions

停止所有程序

pm2 stop all

停止單一程序

pm2 stop 0

pm2 stop [pid]

重啟所有程序

pm2 restart all

刪除單一程序

pm2 delete 0

pm2 delete [pid]
