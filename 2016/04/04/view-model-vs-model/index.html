view model vs model

4月 04, 2016

View Model

除了 MVC 之外，後來 Microsoft 又提出一個 model-view-view model (MVVM)，view model 是一種 model 的變形，有時候你 view 需要的資料，不只會包含一個 model，又或者它僅需要部分的資料，因為有部分不能傳給使用者，例如資料庫中記錄的密碼或一些對使用者無意義的資料，但對系統有意義的資料。
